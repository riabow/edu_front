# Система бесплатных объявлений - Frontend

Vue.js приложение для системы бесплатных объявлений.

## Функционал

- **Главная страница**: Поиск объявлений по:
  - Слову в описании
  - Городу
  - Цене (от/до)
  - Типу объявления
  
- **Личный кабинет**:
  - Просмотр всех своих объявлений
  - Добавление нового объявления
  - Редактирование объявлений
  - Удаление объявлений
  - Загрузка изображений

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` на основе `.env.example` и укажите URL вашего backend:
```
VITE_API_BASE_URL=http://localhost:8000
```

3. Запустите dev сервер:
```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка для production

```bash
npm run build
```

## Структура проекта

```
src/
  ├── api/
  │   └── api.js          # API сервис для работы с backend
  ├── views/
  │   ├── Home.vue        # Главная страница с поиском
  │   ├── Login.vue       # Страница входа
  │   ├── Dashboard.vue   # Личный кабинет
  │   └── ItemForm.vue    # Форма создания/редактирования объявления
  ├── router/
  │   └── index.js        # Настройка роутинга
  ├── App.vue             # Главный компонент
  ├── main.js             # Точка входа
  └── style.css           # Глобальные стили
```

## Используемые технологии

- Vue 3 (Composition API)
- Vue Router 4
- Axios для HTTP запросов
- Vite для сборки
